💡 文件使用提示
═══════════════════════════════════════════════════════════

.bat 文件 (批处理脚本)
  ✓ 双击直接运行
  ✓ 需要 Windows 环境
  ✓ 显示彩色输出

.ps1 文件 (PowerShell 脚本)
  ✓ 运行: .\START_LOCAL.ps1
  ✓ 功能更强大
  ✓ 需要 PowerShell

.txt 文件 (文本文件)
  ✓ 用记事本或任何编辑器打开
  ✓ 格式化输出，易于阅读
  ✓ 离线可查看

.md 文件 (Markdown 文件)
  ✓ 用文本编辑器打开
  ✓ 在 GitHub 上查看效果最好
  ✓ 包含代码示例和格式
