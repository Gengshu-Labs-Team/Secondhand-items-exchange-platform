✨ 项目架构
═══════════════════════════════════════════════════════════════════

客户端 (前端)
├─ 技术: Vue 3 + Vite
├─ 位置: client/ 目录
└─ 特性: 移动端友好界面 (Vant UI)

服务器 (后端)
├─ 技术: Node.js + Express
├─ 位置: server/ 目录
├─ 数据库: SQLite (server/data/campus.db)
└─ 文件存储: server/uploads/

部署方式
├─ 本地: npm start
├─ Docker: docker-compose up -d
└─ 云: 参考 deploy/README.md
